## 自选页面 (Watchlist)

### 文件位置
`web/src/pages/watchlist/WatchlistPage.tsx`

---

#### 变更前 (v1.0)

**布局结构：**
```
自选基金  ← 标题18px，字重600

┌──────────────────────────────────────────────────────┐
│ [基金名称] [代码] [类型] │ ¥12,345 │ +2.34% │ +¥56 │ +¥1,234(5.6%)│ [取消]
└──────────────────────────────────────────────────────┘
  ↑ 完整的持仓信息展示（与持仓页相同）
```

**问题：**
- ❌ 显示了不必要的持仓金额、收益数据
- ❌ 与持仓列表视觉上无差异
- ❌ 取消按钮始终可见，占用空间
- ❌ 标题字号偏小

---

#### 变更后 (v2.0) ✨

**布局结构：**
```
自选基金  ← 标题22px，字重700

┌────────────────────────────────────────────────┐
│ [基金名称]                      │ +2.34%    │ █ │
│ [代码] [类型] 净值1.2345        │ +0.0289   │ 色│  ← hover时显示[取消]
└────────────────────────────────────────────────┘
  ↑ 精简模式：只显示核心信息 + 渐变色条
```

**改进亮点：**

| 项目 | v1.0 | v2.0 | 说明 |
|------|------|------|------|
| **页面标题** | 18px / 600 | **22px / 700** | 更醒目 |
| **组件模式** | holding (默认) | **watchlist** | 精简显示 |
| **持仓金额** | ✅ 显示 | ❌ 移除 | 聚焦基金本身 |
| **当日收益** | ✅ 显示 | ❌ 移除 | 减少干扰 |
| **累计收益** | ✅ 显示 | ❌ 移除 | 减少干扰 |
| **净值显示** | ❌ 无 | **✅ 新增** | 4位小数精度 |
| **涨幅字号** | 15px | **17px** | 更突出 |
| **渐变色条** | ❌ 无 | **✅ 新增** | 直观识别涨跌 |
| **取消按钮** | 始终可见 | **hover显示** | 更清爽 |
| **按钮位置** | top: 12px | **垂直居中** | 更美观 |

**代码关键差异：**

```tsx
// v1.0 - Watchlist Page
<div style={{ padding: '8px 16px' }}>
  <div style={{ fontSize: 18, fontWeight: 600 }}>自选基金</div>
  {favorites.map((item) => (
    <div key={item.id}>
      <FundListItem fund={item} />  {/* 使用默认holding模式 */}
      <Button style={{ position: 'absolute', right: 8, top: 12 }}>  {/* 始终可见 */}
        取消
      </Button>
    </div>
  ))}
</div>

// v2.0 - Watchlist Page ✨
<div style={{ padding: '20px 0' }}>  {/* 增加上下内边距 */}
  <div style={{
    padding: '0 16px 12px',
    fontSize: 22,
    fontWeight: 700,  /* 增强 */
    color: 'var(--text-primary)',
    letterSpacing: '-0.01em',
  }}>
    自选基金
  </div>
  {favorites.map((item) => (
    <div key={item.id} style={{ position: 'relative', marginBottom: 2 }}>
      <FundListItem fund={item} mode="watchlist" />  {/* ✨ 使用watchlist模式 */}
      <Button
        style={{
          position: 'absolute',
          right: 8,
          top: '50%',            /* 垂直居中 */
          transform: 'translateY(-50%)',
          opacity: 0,            /* 默认隐藏 */
          transition: 'opacity var(--transition-fast)',
        }}
        onMouseEnter={(e) => e.currentTarget.style.opacity = '1'}  {/* hover显示 */}
        onMouseLeave={(e) => e.currentTarget.style.opacity = '0'}
      >
        取消
      </Button>
    </div>
  ))}
</div>
```

**视觉效果对比：**

v1.0:
- 信息过载：5列数据密集排列
- 视觉噪音：取消按钮始终占据右侧空间
- 功能混淆：与持仓列表无法区分

v2.0:
- 信息聚焦：只显示基金本身属性（名称、代码、净值、涨幅）
- 视觉清晰：渐变色条提供直观的涨跌感知
- 交互优雅：取消按钮按需显示，界面更清爽

```tsx
// 1. 指数选择器
<Segmented
  value={selectedIndex}
  onChange={(v) => setSelectedIndex(v as IndexCode)}
  options={ALL_INDEX_META.map(meta => ({
    value: meta.code,
    label: meta.nameShort,
  }))}
/>

// 2. 动态主题色
const isUp = (currentIndex?.change ?? 0) >= 0;
<Card style={{
  background: `linear-gradient(135deg, 
    ${isUp ? 'rgba(239,68,68,0.05)' : 'rgba(34,197,94,0.05)'}, 
    rgba(17,24,39,0.8)
  )`,
  borderColor: isUp ? 'rgba(239,68,68,0.15)' : 'rgba(34,197,94,0.15)',
}}>
  {/* 32px超大数字 */}
</Card>

// 3. 自动刷新
useEffect(() => {
  loadIndexData();
  const timer = setInterval(loadIndexData, 60000);  // 60秒
  return () => clearInterval(timer);  // 清理定时器
}, []);
```
