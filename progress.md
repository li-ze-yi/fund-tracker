#### 实现成果

**核心改进**:
1. **纵坐标改为涨跌幅百分比**
   - 新增 `convertToPercentage()` 函数
   - 以起始日为0%基准计算累计收益率
   - Y轴格式：`+12.5%` / `-3.2%`

2. **横坐标动态简化**
   - 实现 `getXAxisConfig()` 函数
   - 根据时间周期自动调整：
     * 近1周: 每天显示
     * 近1月: 每隔2天
     * 近3月: 每4天或月份
     * 近6月/1年: 每6天或双月
     * 全部: 每年

3. **添加0%基准线**
   - markLine 虚线参考线
   - 灰色静默模式

4. **视觉风格统一**
   - 保持墨玉金深色主题
   - 品牌金色线条 (#D4A84B)
   - 半透明渐变面积填充
   - 深色玻璃态 tooltip

5. **交互功能保留**
   - 时间周期切换 ✓
   - 买入/卖出点标记 ✓
   - 缩放拖拽功能 ✓
   - Tooltip 增强 ✓

#### 关键技术决策
- 图表库：继续使用 ECharts（项目已集成，功能强大）
- 数据处理：前端转换，不影响后端API
- 坐标系统：买入/卖出点使用索引查找对应百分比值
- 性能：保持原有 dataZoom 和懒加载机制

#### 发现与决策
- 主流基金软件普遍采用收益率百分比作为Y轴
- X轴密度需要根据时间跨度动态调整
- 0%基准线是重要的视觉参考
- 保持品牌一致性比追求花哨效果更重要
