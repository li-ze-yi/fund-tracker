## ✨ 核心改进成果

### 1️⃣ **纵坐标改为涨跌幅百分比** ✓

**实现位置**: [convertToPercentage()](web/src/pages/fund/FundDetailPage.tsx#L99-L114)

**技术细节**:
```typescript
const convertToPercentage = (history: any[]) => {
  const baseNav = history[0]?.nav || history[0]?.net_value || 1;
  return history.map((d) => {
    const currentNav = d.nav || d.net_value || d.净值 || baseNav;
    return ((Number(currentNav) - baseNav) / baseNav) * 100;
  });
};
```

**用户体验提升**:
- 直观展示收益表现（+12.5% 或 -3.2%）
- 符合主流基金软件习惯（天天基金、支付宝等）
- 方便不同基金之间的收益对比
